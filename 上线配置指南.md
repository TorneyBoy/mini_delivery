# 微信小程序上线配置指南

本文档详细说明了微信小程序配送管理系统上线前需要完成的所有配置步骤。

---

## 一、微信公众平台配置

### 1.1 注册微信小程序账号

1. 访问 [微信公众平台](https://mp.weixin.qq.com/)
2. 注册小程序账号（如已有可跳过）
3. 完成主体认证（企业/个体工商户）

### 1.2 获取小程序基本信息

登录微信公众平台后，在「开发管理」-「开发设置」中获取：

| 配置项    | 说明       | 配置位置                      |
| --------- | ---------- | ----------------------------- |
| AppID     | 小程序ID   | 开发设置 → 开发者ID           |
| AppSecret | 小程序密钥 | 开发设置 → 开发者ID（需生成） |

### 1.3 配置服务器域名

在「开发管理」-「开发设置」-「服务器域名」中配置：

**request合法域名：**
```
https://your-domain.com
```

**注意事项：**
- 必须使用HTTPS协议
- 域名需要ICP备案
- 不支持IP地址和端口号

## 二、微信支付配置

### 2.1 开通微信支付商户号

1. 访问 [微信支付商户平台](https://pay.weixin.qq.com/)
2. 注册商户号并完成企业认证
3. 关联小程序AppID

### 2.2 获取支付配置信息

在商户平台获取以下信息：

| 配置项         | 说明              | 获取位置                       |
| -------------- | ----------------- | ------------------------------ |
| MchID          | 商户号            | 账户中心 → 商户信息            |
| APIv3密钥      | API安全密钥       | 账户中心 → API安全 → APIv3密钥 |
| 商户证书序列号 | 证书序列号        | 账户中心 → API安全 → API证书   |
| 商户私钥       | apiclient_key.pem | API证书下载后解压获取          |

### 2.3 生成API证书

1. 登录商户平台
2. 进入「账户中心」-「API安全」
3. 申请API证书
4. 下载证书压缩包，解压后获取：
   - `apiclient_key.pem` - 商户私钥（重要！）
   - `apiclient_cert.pem` - 商户证书
   - `apiclient_cert.p12` - 商户证书（PKCS12格式）

### 2.4 配置支付回调地址

在商户平台「产品中心」-「开发配置」中设置：

```
https://your-domain.com/api/pay/notify
```

**注意：** 回调地址必须与实际部署域名一致

## 三、微信订阅消息配置

### 3.1 申请消息模板

1. 登录微信公众平台
2. 进入「功能」-「订阅消息」
3. 申请以下模板：

| 模板用途     | 模板名称建议     | 关键词                     |
| ------------ | ---------------- | -------------------------- |
| 账单生成通知 | 账单生成提醒     | 账单类型、金额、日期、备注 |
| 订单分配通知 | 订单分配提醒     | 店铺名称、日期、备注       |
| 订单状态通知 | 订单状态变更提醒 | 订单类型、状态、日期       |
| 送货完成通知 | 送货完成提醒     | 订单类型、日期、备注       |
| 支付成功通知 | 支付成功提醒     | 业务类型、金额、日期       |

### 3.2 记录模板ID

申请成功后，记录每个模板的模板ID（格式如：`XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX`）

## 四、腾讯地图配置

### 4.1 申请腾讯地图Key

1. 访问 [腾讯位置服务](https://lbs.qq.com/)
2. 注册/登录账号
3. 进入「控制台」-「应用管理」-「我的应用」
4. 创建应用，添加Key
5. 选择「WebService API」服务

### 4.2 配置Key

记录生成的Key（格式如：`XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX`）

## 五、后端服务配置

### 5.1 配置文件修改

修改 `server/src/main/resources/application.yml`：

```yaml
# 微信小程序配置
wechat:
  miniapp:
    appid: wx1234567890abcdef        # 替换为你的小程序AppID
    secret: your-app-secret           # 替换为你的小程序AppSecret
    token: your-token                 # 消息推送Token（可选）
    aesKey: your-aes-key              # 消息加密Key（可选）
  pay:
    appid: wx1234567890abcdef         # 小程序AppID
    mchId: 1234567890                 # 商户号
    apiV3Key: your-api-v3-key         # APIv3密钥（32位）
    privateKeyPath: classpath:apiclient_key.pem  # 私钥文件路径
    merchantSerialNumber: XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX  # 商户证书序列号
    notifyUrl: https://your-domain.com/api/pay/notify  # 支付回调地址
  # 订阅消息模板ID
  message:
    template:
      bill-created: XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
      order-assigned: XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
      order-status: XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
      delivery-complete: XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
      payment-success: XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
```

### 5.2 上传商户私钥

将下载的 `apiclient_key.pem` 文件放到：
```
server/src/main/resources/apiclient_key.pem
```

### 5.3 数据库更新

执行以下SQL为shop和driver表添加openid字段：

```bash
mysql -u root -p delivery < server/src/main/resources/db/add_openid.sql
```

## 七、服务器部署

### 7.1 环境要求

- JDK 17+
- MySQL 8.0+
- Redis 6.0+（可选，用于缓存）
- Nginx（用于HTTPS和反向代理）

### 7.2 打包后端项目

```bash
cd server
mvn clean package -DskipTests
```

生成的jar包位于：`server/target/mini-delivery-1.0.0.jar`

### 7.3 启动后端服务

```bash
java -jar mini-delivery-1.0.0.jar \
  --spring.datasource.url=jdbc:mysql://localhost:3306/delivery \
  --spring.datasource.username=root \
  --spring.datasource.password=your-password
```

### 7.4 Nginx配置示例

```nginx
server {
    listen 443 ssl;
    server_name your-domain.com;

    ssl_certificate /path/to/cert.pem;
    ssl_certificate_key /path/to/key.pem;

    location /api {
        proxy_pass http://127.0.0.1:8081;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
```

## 八、小程序上传与审核

### 8.1 上传代码

1. 打开微信开发者工具
2. 点击「上传」按钮
3. 填写版本号和项目备注

### 8.2 提交审核

1. 登录微信公众平台
2. 进入「版本管理」
3. 将开发版本提交审核
4. 填写审核信息

### 8.3 发布上线

审核通过后，点击「发布」即可上线

## 十、常见问题

### Q1: 支付回调失败
- 检查回调地址是否正确配置
- 确认服务器HTTPS证书有效
- 查看后端日志确认是否收到回调

### Q2: 订阅消息发送失败
- 确认用户已授权订阅消息
- 检查模板ID是否正确
- 确认openid是否正确获取

### Q3: 地图导航不工作
- 确认腾讯地图Key已配置
- 检查Key是否有WebService API权限
- 确认域名白名单已配置

### Q4: 登录失败
- 检查AppID和AppSecret是否正确
- 确认服务器域名已配置
- 查看后端日志排查问题
