# 微信小程序配送管理系统 - 开发计划

## 一、项目概述

### 1.1 项目背景
开发一款微信小程序，用于货车司机向水果店和超市配送货品的管理。系统涉及四方角色，实现从店铺下单、订单分配、司机拣货到配送完成的完整流程管理。

### 1.2 角色定义
| 角色   | 描述               | 核心功能                               |
| ------ | ------------------ | -------------------------------------- |
| 总管理 | 系统唯一超级管理员 | 分管理账号管理、全局数据监控           |
| 分管理 | 品牌方管理员       | 店铺/司机/商品管理、订单监控、账单生成 |
| 店铺   | 水果店/超市        | 商品浏览、下单、订单管理、账单支付     |
| 司机   | 货车司机           | 订单分配、拣货、送货、导航             |

## 二、技术架构

### 2.1 技术选型

#### 前端（微信小程序）
- **框架**: 原生微信小程序 / Taro / uni-app
- **UI组件库**: Vant Weapp / WeUI
- **图表库**: ECharts for Weixin
- **地图导航**: 腾讯地图 / 高德地图 SDK

#### 后端
- **语言**: Java (Spring Boot 3.x)
- **数据库**: MySQL 8.0（主数据） + Redis（缓存/会话）
- **ORM**: MyBatis-Plus
- **文件存储**: 阿里云OSS / 七牛云 CDN（商品图片）
- **定时任务**: Spring Scheduled
- **安全框架**: Spring Security + JWT

#### 第三方服务
- **微信支付**: 微信支付API
- **消息通知**: 微信订阅消息

## 三、数据库设计

### 3.1 核心数据表

```sql
-- 1. 总管理表
CREATE TABLE admin (
    id INT PRIMARY KEY AUTO_INCREMENT,
    phone VARCHAR(20) UNIQUE NOT NULL,
    password VARCHAR(255) NOT NULL,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- 2. 分管理表
CREATE TABLE branch_manager (
    id INT PRIMARY KEY AUTO_INCREMENT,
    phone VARCHAR(20) UNIQUE NOT NULL,
    password VARCHAR(255) NOT NULL,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    status TINYINT DEFAULT 1 COMMENT '1-正常 0-禁用'
);

-- 3. 店铺表
CREATE TABLE shop (
    id INT PRIMARY KEY AUTO_INCREMENT,
    branch_manager_id INT NOT NULL,
    name VARCHAR(100) NOT NULL,
    address VARCHAR(255) NOT NULL,
    phone VARCHAR(20) NOT NULL,
    password VARCHAR(255) NOT NULL,
    show_price TINYINT DEFAULT 1 COMMENT '是否显示单价',
    price_type TINYINT DEFAULT 1 COMMENT '1-老用户价 2-新用户价',
    status TINYINT DEFAULT 1,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (branch_manager_id) REFERENCES branch_manager(id)
);

-- 4. 司机表
CREATE TABLE driver (
    id INT PRIMARY KEY AUTO_INCREMENT,
    branch_manager_id INT NOT NULL,
    name VARCHAR(50) NOT NULL,
    phone VARCHAR(20) NOT NULL,
    password VARCHAR(255) NOT NULL,
    status TINYINT DEFAULT 1,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (branch_manager_id) REFERENCES branch_manager(id)
);

-- 5. 商品表
CREATE TABLE product (
    id INT PRIMARY KEY AUTO_INCREMENT,
    branch_manager_id INT NOT NULL,
    name VARCHAR(100) NOT NULL,
    category VARCHAR(50),
    old_price DECIMAL(10,2) COMMENT '老用户价',
    new_price DECIMAL(10,2) COMMENT '新用户价',
    unit VARCHAR(20) COMMENT '单位：斤/箱',
    description TEXT,
    image_url VARCHAR(500),
    status TINYINT DEFAULT 1,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (branch_manager_id) REFERENCES branch_manager(id)
);

-- 6. 订单表
CREATE TABLE `order` (
    id INT PRIMARY KEY AUTO_INCREMENT,
    order_no VARCHAR(50) UNIQUE NOT NULL,
    shop_id INT NOT NULL,
    driver_id INT COMMENT '分配的司机ID',
    status TINYINT DEFAULT 0 COMMENT '0-待支付 1-待分配 2-待拣货 3-待送货 4-已完成',
    total_amount DECIMAL(10,2) NOT NULL,
    delivery_date DATE NOT NULL COMMENT '收货日期',
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    received_at DATETIME COMMENT '收货时间',
    FOREIGN KEY (shop_id) REFERENCES shop(id),
    FOREIGN KEY (driver_id) REFERENCES driver(id)
);

-- 7. 订单商品关联表
CREATE TABLE order_item (
    id INT PRIMARY KEY AUTO_INCREMENT,
    order_id INT NOT NULL,
    product_id INT NOT NULL,
    quantity DECIMAL(10,2) NOT NULL,
    unit_price DECIMAL(10,2) NOT NULL COMMENT '下单时的单价',
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (order_id) REFERENCES `order`(id),
    FOREIGN KEY (product_id) REFERENCES product(id)
);

-- 8. 账单表
CREATE TABLE bill (
    id INT PRIMARY KEY AUTO_INCREMENT,
    bill_no VARCHAR(50) UNIQUE NOT NULL,
    shop_id INT NOT NULL,
    bill_date DATE NOT NULL COMMENT '账单日期',
    total_amount DECIMAL(10,2) NOT NULL,
    status TINYINT DEFAULT 0 COMMENT '0-待支付 1-已支付',
    paid_at DATETIME COMMENT '支付时间',
    wechat_transaction_id VARCHAR(100) COMMENT '微信交易号',
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (shop_id) REFERENCES shop(id)
);

-- 9. 账单订单关联表
CREATE TABLE bill_order (
    id INT PRIMARY KEY AUTO_INCREMENT,
    bill_id INT NOT NULL,
    order_id INT NOT NULL,
    FOREIGN KEY (bill_id) REFERENCES bill(id),
    FOREIGN KEY (order_id) REFERENCES `order`(id)
);

-- 10. 拣货清单表
CREATE TABLE picking_list (
    id INT PRIMARY KEY AUTO_INCREMENT,
    driver_id INT NOT NULL,
    product_id INT NOT NULL,
    total_quantity DECIMAL(10,2) NOT NULL,
    picked_quantity DECIMAL(10,2) DEFAULT 0,
    status TINYINT DEFAULT 0 COMMENT '0-待拣货 1-已完成',
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (driver_id) REFERENCES driver(id),
    FOREIGN KEY (product_id) REFERENCES product(id)
);

-- 11. 送货清单表
CREATE TABLE delivery_list (
    id INT PRIMARY KEY AUTO_INCREMENT,
    driver_id INT NOT NULL,
    shop_id INT NOT NULL,
    status TINYINT DEFAULT 0 COMMENT '0-待送货 1-已完成',
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    completed_at DATETIME,
    FOREIGN KEY (driver_id) REFERENCES driver(id),
    FOREIGN KEY (shop_id) REFERENCES shop(id)
);

-- 12. 送货清单订单关联表
CREATE TABLE delivery_order (
    id INT PRIMARY KEY AUTO_INCREMENT,
    delivery_list_id INT NOT NULL,
    order_id INT NOT NULL,
    FOREIGN KEY (delivery_list_id) REFERENCES delivery_list(id),
    FOREIGN KEY (order_id) REFERENCES `order`(id)
);

-- 13. 自动任务日志表
CREATE TABLE task_log (
    id INT PRIMARY KEY AUTO_INCREMENT,
    task_type VARCHAR(50) NOT NULL COMMENT '任务类型',
    executed_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    status TINYINT COMMENT '0-失败 1-成功',
    detail TEXT COMMENT '执行详情'
);
```
